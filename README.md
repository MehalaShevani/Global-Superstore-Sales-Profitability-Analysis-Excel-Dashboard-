# Global-Superstore-Sales-Profitability-Analysis-Excel-Dashboard-

This project analyzes the Global Superstore retail dataset to evaluate sales performance, profitability drivers, discount impact and regional trends using Excel.  The Objective is not just to build charts, but to transform transactional data into business insights that support strategic decision-making.

### Business Questions Answered

* Are sales growing over time?
* Which product categories drive the most profit?
* Does heavy discounting erode margins?
* Are high-sales regions also profitable?
* Is revenueconcentrated among a small group of customers?

### Tools Used

-> Microsoft Excel
-> Pivot Tables
-> Pivot Charts
-> Feature Engineering
-> Data cleaning & Formatting
-> Interactive Slicers

### Key Anlaytical Steps

1. Data Preparation
   Removed duplicates
   Standardized date and numeric formats
   Preserved raw dataset separately
   Converted cleaned data into a structured Excel Table
2. Feature Engineering
   Created business relevant metrics including:
   Profit Margin
   Discount Bands
   Shipping Days
   Monthly Time Aggregation(via grouped date fields)
3. Pivot Based Analysis
   Monthly sales trends
   Profit by category
   Regional performance
   Discount impact on margin
   Customer contribution
4. Interactive Dashboard
   KPI cards(Total Sales, Total Profit, Average Profit Margin)
   Trend anaysis
   Profit drivers
   Interactive slicers

### Key Insights

* Sales show consistent growth with seasonal variation.
* Technology category contributes the highest profit share.
* Higher discount levels correlate with reduced profit margins.
* A small segment of customers contributes disproportionately to total profit.
* Some regions generate high sales volume without equivalent profitability.
